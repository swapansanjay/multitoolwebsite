A child theme for Astra that provides a collection of unit converters and calculators.

== Description ==

Unit Converters Child Theme is a specialized WordPress theme built on top of Astra, designed to showcase and provide various unit conversion tools and calculators. The theme features a modern, responsive design with Bootstrap 5 integration and a user-friendly interface.

== Features ==

* Responsive design with Bootstrap 5
* Custom post type for tools
* Custom navigation walker for Bootstrap 5 menus
* Theme customizer options for:
  * Social media links
  * Contact information
  * Theme colors
  * Advertisement settings
* Mobile-friendly layout
* SEO optimized structure

== Installation ==

1. Upload the theme files to the `/wp-content/themes/unit-converters-child` directory, or install the theme through the WordPress themes screen directly.
2. Activate the theme through the 'Themes' screen in WordPress.
3. Use the Customizer to configure the theme settings.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Unit Converters Child Theme is compatible with most WordPress plugins, but it's specifically designed to work with:
* Astra (parent theme)
* Bootstrap 5
* WordPress core features
